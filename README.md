### 🏠 [Início](https://github.com/thiagojacinto/virada-qualidade-desafio#readme)

## Install

```sh
npm install
```

## Usage

```sh
npm run cypress
```

## Run tests

```sh
npm test
```
### Run UI tests

```sh
npm run test:ui
```
### Run API tests

```sh
npm run test:api
```
### Run all tests  and generate a report

```sh
npm run test:report
```
> After generated, reports should be found at _cypress/reports_
